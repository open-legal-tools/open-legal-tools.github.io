# Open Legal Tools Website

This repository contains the official website for the Open Legal Tools organization, showcasing our mission, projects, and roadmap.

## About Open Legal Tools

Open Legal Tools creates professional-grade, open-source software that makes legal documents more accessible, navigable, and useful for everyone—from judicial clerks to the public. Our first product, Gavel, is a sophisticated document navigator that transforms how legal professionals interact with case documents.

## Website Structure

The website is built with clean, semantic HTML, CSS, and minimal JavaScript to ensure accessibility and performance. The site consists of the following main sections:

- **Home**: Overview of Open Legal Tools and Gavel
- **About**: Our mission, values, and team information
- **Products**: Detailed information about Gavel
- **Roadmap**: Development timeline and upcoming features
- **License**: Information about our AGPL v3.0 licensing
- **Contact**: How to reach us and get involved

## Getting Started

### Prerequisites

This is a static website that requires no special tools for development beyond:
- A code editor (VS Code, Sublime Text, etc.)
- Git
- A modern web browser

### Local Development

To work on this website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/open-legal-tools/open-legal-tools.github.io.git
   cd open-legal-tools.github.io
   ```

2. Open `index.html` in your browser to view the site.

3. Make your changes to the HTML, CSS, or JavaScript files.

4. Refresh your browser to see the changes.

### Deployment

This website is automatically deployed using GitHub Pages whenever changes are pushed to the main branch.
